### File format detail: .rtf (dictionary)

Magic bytes / header:
```
{\rtf1\ansi{\*\cxrev100}{\*\cxdict}
{\*\cxsystem caseCATalyst4 version 22.13 by Stenograph L.L.C.}
```
The `{\*\cxdict}` marker identifies this as a CRE/RTF dictionary export.
Entries follow the pattern `{\*\cxs STENO_STROKES}translation_text`.
Sample entries:
```
{\*\cxs STPUPBD/-G}funding
{\*\cxs STORPL/HARB/OR/SKAOURTS}Storm Harbour Securities
{\*\cxs SALT/PHAPB}Saltsman
{\*\cxs SKWREUPL/KWREUPL}\s4 MR. GARNER:
```
