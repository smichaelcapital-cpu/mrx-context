## Defect Description

The INDEX page slot 1 is rendered as:

```
  1  I N D E X
```

Oracle shows slot 1 as a two-position slot:

```
  1  I N D E X          <- MAIN
     Page               <- SUB
```

The SUB position of slot 1 holds the column header `Page`, which labels
the page-number column that appears to the right of each index entry
throughout the INDEX section.

This `Page` header is absent from the renderer output across all depos.
It is a universal cosmetic defect within the MB front-matter profile.

## Implementation Notes

- The INDEX page header is rendered in `src/stage5/front_matter/index.py`.
- Locate the line that emits slot 1 (`I N D E X`). It currently emits only
  the MAIN content. Add the SUB emission for `Page` to the same slot.
- The `Page` string must match the oracle exactly (plain `Page`, no period,
  no indent beyond what the SUB column formatting requires).
- Do NOT insert a new slot. The fix is one additional SUB line within the
  existing slot 1 emission.
